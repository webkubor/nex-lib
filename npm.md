

镜像站



## 切换镜像源
```
npm config set registry https://registry.npmjs.org/
npm config set registry https://npmmirror.com/
```
##  发布到 npm
```
npm  login
npm publish
npm info nex-lib
```


## 清空代理
```
npm config set proxy null
npm config set https-proxy null
```


## 移除 npm 包
```
npm unpublish XXXXX  --force
```
